<h1 align="center" id="title">Sparql-RDF4J End Point</h1>

<p id="description">This project provides a lightweight SPARQL Endpoint Executor designed to interact seamlessly with an RDF4J Server and multiple federated SPARQL endpoints. It enables users to submit SPARQL queries route them to local or remote repositories and retrieve results in a unified and efficient workflow. The executor supports SELECT CONSTRUCT ASK and UPDATE operations making it suitable for testing integrating and extending semantic web applications that rely on distributed RDF data sources.</p>
